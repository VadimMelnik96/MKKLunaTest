# Задание для МКК Луна

## Запуск приложения

1. Создать .env в корне проекта и скопировать содержимое .env.example в .env
2. Ввести команду 
```
docker compose up --build 
```
3. API будет доступно по адресу http://127.0.0.1:8000/docs
4. В плашке Authorize ввести значение APP_STATIC_API_KEY из .env
5. Реализован тестовый роутер с эндпоинтом test/flood/{amount}, который генерирует заданное количество платежей и шлет на /payments, для демонстрации работы DLQ рекомендую генерировать сразу в районе 300 
6. Интерфейс RabbitMQ доступен по адресу http://localhost:15672/ , username и пароль также можно взять из .env
7. Idempotency-key можно проверить, например, поочередно введя тестовые данные: одну и ту же строку в хэдер Idempotency-Key и, например, два разных платежа:
```
{
  "amount": 100,
  "currency": "USD",
  "description": "important_payment",
  "payment_metadata": {
    "type": "withdraw"
  },
  "webhook_url": "https://www.google.com/"
}
```

и затем

```
{
  "amount": 200.34,
  "currency": "EUR",
  "description": "Pogovorim o nagrade",
  "payment_metadata": {
    "sender": "Geralt from Rivia"
  },
  "webhook_url": "https://www.google.com/"
}
```
